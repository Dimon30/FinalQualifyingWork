# Архив: симуляции Глав 2 и 3

Рабочий архив симуляций **Глав 2 и 3** диссертации Ким С.А. (2024).

Код полностью работоспособен. Перенесён в архив, поскольку основной фокус
проекта — согласованное траекторное управление по выходу (Глава 4).

## Содержание

| Файл | Описание |
|------|----------|
| `run_ch2.py` | Гл. 2: стабилизация в точке x*=(1, 0.5, 2), φ*=0 |
| `run_ch2_path.py` | Гл. 2: движение по ломаной A(5,5,5)→B→C→D(30,20,20) |
| `run_ch3.py` | Гл. 3: следящее управление по выходу, спираль r=0.5 |
| `run_all.py` | Запуск всех глав (2, 3, 4) разом |
| `trajectories.py` | Временна́я параметризация опорной траектории (Гл. 3) |
| `controllers/ch2_state.py` | Регулятор Гл. 2 (управление по состоянию) |
| `controllers/ch3_output.py` | Регулятор Гл. 3 (следящее управление по выходу) |

## Как запустить

Скрипты ожидают, что `code/` находится в `sys.path`. Из корня проекта:

```bash
# Добавить code_app/ в PYTHONPATH и запустить
PYTHONPATH=code_app python legacy/code_ch2_ch3_04_13_archive/run_ch2.py
PYTHONPATH=code_app python legacy/code_ch2_ch3_04_13_archive/run_ch3.py
```

Или скопировать нужные файлы в `code/` и запускать оттуда.
