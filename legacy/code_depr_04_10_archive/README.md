# Python-симуляции по диссертации (квадрокоптер)

Этот пакет собирает **три режима**, которые соответствуют трём «ступеням» диссертации:

1) Глава 2 — стабилизация в точке (позиция + yaw).
2) Глава 3 — следящее управление (траектория как функция времени), через динамическое расширение и регулятор в нормальной форме.
3) Глава 4 — **согласованное** управление вдоль кривой (траектория как множество точек), с параметром ближайшей точки `s`
   и поперечными отклонениями `e1,e2`, и требованием `s_dot → V*`.

⚠️ Важно: в диссертации обозначения углов нестандартные:
- φ — yaw (рысканье)
- θ — pitch (тангаж)
- ψ — roll (крен)

## Установка

```bash
pip install numpy scipy matplotlib
```

## Запуск демо
```bash

```
