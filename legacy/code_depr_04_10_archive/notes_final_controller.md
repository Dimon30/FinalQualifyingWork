# Итоговый (самый общий) закон управления в диссертации — Глава 4

В диссертации задачи траекторного управления разделены на **следящее** и **согласованное** управление (Глава 4).
Ключевой результат — синтез **согласованного управления по выходу** для движения вдоль гладкой пространственной кривой
по измерениям/оценкам только отклонений `s, e1, e2, δ` (где `δφ = φ − φ*(s)`), а не по заранее заданному времени.

Регулируемые переменные (выход):
- `s = Σ(P)` — параметр ближайшей к роботу точки на кривой S,
- `e1 = E1(P)`, `e2 = E2(P)` — поперечные отклонения от двух плоскостей,
- `δφ = φ − φ*(s)` — ошибка ориентации (yaw относительно касательной).

Цель в постановке: ограничить боковое отклонение и ориентацию, и добиться `ṡ → V*`.

## Почему это «круто»
1) **Не требует параметризации по времени**: траектория — кривая, робот всегда «цепляется» за ближайшую точку.
2) **Согласование скорости**: регулятор стремит `ṡ` к `V*`, а не просто «догоняет точку».
3) **Робастность/неполная информация**: управление по выходу + расширенный наблюдатель производных; допускается ограниченная ошибка,
   величина которой настраивается.
4) **Ограничения входов**: используется насыщение `sat_L(·)`.

## Структура итогового закона (по диссертации)
В нормальной форме (цепочка интеграторов до 4-й производной выхода)
`ξ1=[s,e1,e2,δφ]^T, ξ2=ξ̇1, ξ3=ξ̇2, ξ4=ξ̇3`, после динамического расширения по `u1,u2`:

`U = sat_L( b(·)^{-1} W(α,β)^{-1} ( -σ - γ1 ξ̂1 - γ2 ξ̂2 - γ3 ξ̂3 - γ4 ξ̂4 ) )`.

`ξ̂i` и `σ` — состояния расширенного (high-gain) наблюдателя.

В этом архиве итоговый регулятор реализован в `controllers/ch4_coordinated.py`.

## Итоговая формула (как в тексте диссертации)

**Динамическое расширение**:

```
u1 = sat_L(ū1),
ū1¨ = v̄1,
u2¨ = v2,

[v̄1, v2, u3, u4]^T = γ5 η + Ū,
η̇ = Ū.
```

**Управление по выходу** (инвертирование канала через `W` и `b` + насыщение):

```
Ū = sat_L( b(θ,ψ,u1,φ)^{-1} W(α,β)^{-1} ( -σ - γ1 λ̂1 - γ2 λ̂2 - γ3 λ̂3 - γ4 λ̂4 ) ).
```

**Расширенный наблюдатель производных** (оценка λ̂i и σ):

```
λ̂̇1 = λ̂2 + κ a1 (λ̃1 − λ̂1)
λ̂̇2 = λ̂3 + κ^2 a2 (λ̃1 − λ̂1)
λ̂̇3 = λ̂4 + κ^3 a3 (λ̃1 − λ̂1)
λ̂̇4 = σ + W b Ū + κ^4 a4 (λ̃1 − λ̂1)
σ̇   = κ^5 a5 (λ̃1 − λ̂1)
```

В отличие от «следящего» трекинга по времени, здесь цель — **согласование скорости вдоль кривой**
(т.е. `ṡ → V*`) и стабилизация боковых отклонений `e1,e2` и ошибки ориентации `δφ`.
